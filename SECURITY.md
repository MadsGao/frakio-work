# Security Policy

Security fixes are provided for the latest published beta.

Please report vulnerabilities through GitHub private vulnerability reporting for `MadsGao/frakio-work`. Do not include real API keys, OAuth tokens, conversation content, or private local paths in a public issue.

Frakio Work exposes a loopback-only management API. Reports involving Origin validation, local session bypass, workspace path escape, unsafe command execution, secret exposure, or release-update redirection are especially useful.
